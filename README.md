# INSTAGRAM Workspace

This workspace contains a React + Firebase Instagram clone application.

## App Location

- Main app: `Instagram`

## Quick Start

1. Open terminal in `Instagram`
2. Install dependencies
3. Create `.env` from `.env.example`
4. Start development server

Commands:

- `cd Instagram`
- `npm install`
- `copy .env.example .env` (PowerShell)
- `npm run dev`
